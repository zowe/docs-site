## Description
Documentation states that MEMLIMIT should not be set, because it may hide issues: "Setting a MEMLIMIT is not recommended as doing so prevents issues caused by insufficient memory."
Users disagree with not setting the limit, and this statement seems to be incorrect. Actually, not setting the limit probably prevents the issues caused by insufficient memory.
