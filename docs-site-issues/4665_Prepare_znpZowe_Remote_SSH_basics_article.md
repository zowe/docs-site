## Is your request for enhancement related to a problem? Please describe.
With the Zowe Remote SSH extension approaching GA status, users would benefit from having some instructions on how to use the extension.

## Describe the solution you'd like
We should prepare a page in Zowe Docs (similar to what we have for the FTP extension) that tells users how to:
- Install the extension
- Set up the connection and deploy the server executables
- Troubleshoot the extension

We should identify any necessary configuration/access settings that need to be made in order for users to be able to launch and communicate with the znp server (more research needed).
