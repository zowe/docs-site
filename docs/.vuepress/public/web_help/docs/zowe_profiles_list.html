zowe → profiles → list

List profiles of the type {{type}}

Usage

zowe profiles list <command>

Where <command> is one of the following:

Commands

  • cics-profiles | cics - List profiles of the type cics
  • db2-profiles | db2 - List profiles of the type db2
  • ssh-profiles | ssh - List profiles of the type ssh
  • tso-profiles | tso - List profiles of the type tso
  • zosmf-profiles | zosmf - List profiles of the type zosmf
