List the z/OSMF workflows for a system or a sysplex with filter options.

Usage

zowe zos-workflows list <command>

Where <command> is one of the following:

Commands

  • active-workflow-details | awd - Get the details of an active z/OSMF workflow
  • active-workflows | aw - List active workflow instance(s) in z/OSMF.
  • archived-workflows | arw - List all archived workflows for a system.
  • definition-file-details | dfd - Retrieve the contents of a z/OSMF workflow definition from a z/OS system.
