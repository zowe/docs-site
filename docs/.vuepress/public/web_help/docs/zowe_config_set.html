Set a configuration setting.

Usage

zowe config set <configName> <configValue> [options]

Positional Arguments

  • configName (string)

    • Setting name. Possible values: CredentialManager - The package name of a plugin that will override the default credential manager to allow for different credential storage methods.

  • configValue (string)

    • Value to set

Examples

  • Set the default credential manager to my-credential-manager:

    • zowe config set CredentialManager my-credential-manager
