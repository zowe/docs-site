Delete resources (for example, programs) from CICS through IBM CMCI.

Usage

zowe cics delete <command>

Where <command> is one of the following:

Commands

  • program | prog - Delete a program from CICS.
  • transaction | tran - Delete a transaction from CICS.
  • urimap - Delete a urimap from CICS.
  • webservice | web - Delete a web service from CICS.
