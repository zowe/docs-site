zowe → profiles → create

Create new configuration profiles.

Usage

zowe profiles create <command>

Where <command> is one of the following:

Commands

  • cics-profile | cics - Create a cics profile
  • db2-profile | db2 - Create a db2 profile
  • ssh-profile | ssh - Create a ssh profile
  • tso-profile | tso - Create a tso profile
  • zosmf-profile | zosmf - Create a zosmf profile
