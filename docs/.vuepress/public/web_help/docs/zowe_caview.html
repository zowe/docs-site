Access CA View data and perform related tasks.

Usage

zowe caview <group>

Where <group> is one of the following:

Commands

  • download | dl - Download CA View report data to local folder.
  • export | exp - Export CA View text report file to local folder.
  • get - Get information from CA View repository.
  • list | ls - List CA View repository, report, and user information.
  • search | find - Search data in a CA View repository.
  • set - Set information in CA View repository.
