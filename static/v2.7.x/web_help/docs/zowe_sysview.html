Zowe CLI plugin for SYSVIEW

Usage

zowe sysview <command|group>

Where <command|group> is one of the following:

Commands

  • capture | cap - Display a captured SYSVIEW primary command
  • display | dsp - Display a real-time SYSVIEW primary command
  • execute | exe - Execute a real-time SYSVIEW function command

Groups

  • check - Confirm that the SYSVIEW REST server is running on a specified system.
