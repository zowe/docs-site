Manage connections to Endevor Web Services used by Endevor Bridge for Git.

Usage

zowe endevor-bridge-for-git endevor-connection <command>

Where <command> is one of the following:

Commands

  • create | c - Create new Endevor connection
  • delete | del - Delete existing Endevor connection
  • list | ls - List Endevor connections
  • rename | ren - Rename existing Endevor connection
  • update-credentials | uc - Update your Endevor credentials for the Endevor connection.
