Install resources (for example, programs) to CICS through IBM CMCI.

Usage

zowe cics install <command>

Where <command> is one of the following:

Commands

  • program | prog - Install a program to CICS.
  • transaction | tran - Install a transaction to CICS.
  • urimap - Install a urimap to CICS.
