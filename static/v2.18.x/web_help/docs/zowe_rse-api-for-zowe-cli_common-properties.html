Manage Common Properties on z/OS RSE API server.

Usage

zowe rse-api-for-zowe-cli common-properties <command>

Where <command> is one of the following:

Commands

  • copy | cp - Copy a namespace
  • create | cre - Create a namespace
  • delete | del - Delete the namespace or a property in the namespace
  • list | ls - List all namespaces
  • rename - Rename a namespace
  • set - Set a namespace property
  • view | vw - View the namespace or a property in the namespace
