Search for PDS member(s) provided the filter with a member pattern and/or content string.

Usage

zowe rse-api-for-zowe-cli search members <filter> <memberPattern> [options]

Positional Arguments

  • filter (string)

    • Dataset filter string. Ex. FILTERNAME.*.

  • memberPattern (string)

    • Member name pattern. Ex. MEM or *

Options

  • --content | -c (string)

    • Content string to be searched. Ex. HelloWorld or *

  • --regex-member | --rm (boolean)

    • Regex for member name. If Regex pattern is not used, wildcard patterns are used.

      Default value: false

  • --regex-content | --rc (boolean)

    • Regex for content. If Regex pattern is not used, wildcard patterns are used.

      Default value: false

  • --case-sensitive-content | --csc (boolean)

    • Case-sensitive for content

      Default value: false

  • --number-of-results | --nor (string)

    • Number of results displayed for MVS search If number-of-results is not used, default is 10000 results.

  • --response-format-type | --rft (string)

    • The command response output format type. Must be one of the following: table, list, object, string

      Default value: list

Profile Options

  • --rse-profile | --rse-p (string)

    • The name of a (rse) profile to load for this command execution.

  • --base-profile | --base-p (string)

    • The name of a (base) profile to load for this command execution.

Base Connection Options

  • --host | -H (string)

    • Host name of service on the mainframe.

  • --port | -P (number)

    • Port number of service on the mainframe.

  • --user | -u (string)

    • User name to authenticate to service on the mainframe.

  • --password | --pass | --pw (string)

    • Password to authenticate to service on the mainframe.

  • --reject-unauthorized | --ru (boolean)

    • Reject self-signed certificates.

      Default value: true

  • --token-type | --tt (string)

    • The type of token to get and use for the API. Omit this option to use the default token type, which is provided by 'zowe auth login'.

  • --token-value | --tv (string)

    • The value of the token to pass to the API.

  • --cert-file (local file path)

    • The file path to a certificate file to use for authentication.

      Note: The CLI does not support certificate files that require a password. For more information, search Troubleshooting PEM Certificates in Zowe Docs.

  • --cert-key-file (local file path)

    • The file path to a certificate key file to use for authentication

Examples

  • Search for a Dataset Member with a filter of USER.* with member pattern TEST:

    • zowe rse-api-for-zowe-cli search members "USER.*" "MEMBER"
