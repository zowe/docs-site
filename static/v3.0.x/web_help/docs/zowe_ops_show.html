Display data associated with OPS/MVS automation elements, such as AOF rules and rulesets, and SSM resources, OPS log names, subsystems and RDF tables, or REST API server status.

Usage

zowe ops show <command>

Where <command> is one of the following:

Commands

  • logname - Display data associated with logs.
  • resource - Display data associated with the specified SSM resource.
  • rule - Display data associated with the specified rule.
  • ruleset - Display data associated with one or all rulesets.
  • status - Display data associated with an OPS/MVS REST API server.
  • subsystem - Display a list of the OPS/MVS subsystems.
  • table - Display data associated with tables.
