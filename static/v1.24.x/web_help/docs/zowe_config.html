Manage configuration and overrides. To see all set-able options use "list" command.

Usage

zowe config <command>

Where <command> is one of the following:

Commands

  • get | ge - Get a value of single setting option
  • list | li - List all configuration setting options
  • reset | re - Reset a configuration setting to default value
  • set | se - Set a configuration setting
