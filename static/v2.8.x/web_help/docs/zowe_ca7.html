Welcome to the CA 7 Zowe CLI!

Usage

zowe ca7 <group>

Where <group> is one of the following:

Groups

  • commands | cmds - All the CA 7 commands
  • create-job-definition | cjd - Creates requirements of a job
  • create-resource-definition | crd - Create a Resource Count resource
  • dataset | ds - Dataset management
  • delete-job-definition | djd - Deletes requirements of a job
  • delete-resource-definition | drd - Deletes a Resource Count
  • event-manager-requests | emr - Retrieves a status of a job from CA 7
  • job-definition | jd - All methods of a job definitions
  • job-instance | ji - All methods of a Job Instances
  • list-job-definition | ljd - Retrieves the job definition
  • list-job-history | ljh - Retrieves job history information
  • list-job-instance | lji - Retrieves the instance definition for a job
  • list-resource-definition | lrd - Retrieves a Resource Count Resource Information
  • update-job-definition | ujd - Updates the requirement definition for a job
  • update-job-instance | uji - Updates the instance definition
  • update-resource-definition | urd - Updates the resource count definition
