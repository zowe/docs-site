zowe ‚Ä∫ zos-files ‚Ä∫ compare

Compare the contents of a two data set members on your terminal (stdout).

Usage

zowe zos-files compare <command>

Where <command> is one of the following:

Commands

  • data-set | ds - Compare content of two z/OS data sets
  • local-file-data-set | lf-ds - Compare content of a local file and a z/os dataset
  • local-file-spool-dd | lf-sdd - Compare content of a local file and a spool dd
  • local-file-uss-file | lf-uss - Compare content of a local file and a z/os uss file
  • spool-dd | sdd - Compare content of two spool dds
  • uss-files | uss | uf - Compare the contents of two z/os uss files
