Manage z/OS Connect EE API Requesters

Usage

zowe zosconnect apirequester <command>

Where <command> is one of the following:

Commands

  • delete - Delete an API Requester
  • info - Get detailed information about an API Requester
  • install - Install a new API Requester
  • list - List the API Requesters installed in the server
  • start - Set an API Requester status to started
  • stop - Set an API Requesters status to stopped
  • update - Update an existing API Requester
