Manage NetView automation tables.

Usage

zowe znetview-automation <group>

Where <group> is one of the following:

Groups

  • activate | act - Activate an automation table
  • download | dl - Download automation table statement(s)
  • list | ls - List the automation table statistics
  • test - Run a simulated automation table test for a Canzlog message
  • upload | ul - Upload automation table statement(s)
  • validate | val - Validate an automation table
