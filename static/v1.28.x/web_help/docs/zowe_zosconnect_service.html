Manage z/OS Connect EE Services

Usage

zowe zosconnect service <command>

Where <command> is one of the following:

Commands

  • delete - Delete a Service
  • info - Get detailed information about a Service
  • install - Install a new Service
  • list - List the Services installed in the server
  • start - Set a Services status to started
  • stop - Set a Services status to stopped
  • update - Update an exisiting Service
