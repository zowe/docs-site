List jobs, jobstreams, and resources in plan, list jobstreams in model(database).

Usage

zowe workload-automation list <command>

Where <command> is one of the following:

Commands

  • jobinplan | jip | jp - List the jobs in plan
  • jobstreaminmodel | jsim | jsm - List the jobstreams in model
  • jobstreaminplan | jsip | jsp - List the jobstreams in plan
  • resourceinplan | resp | rp - List the resources in plan
