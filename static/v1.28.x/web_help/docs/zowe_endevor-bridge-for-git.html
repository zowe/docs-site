Use the CA Endevor Bridge for Git plug-in to manage your Git-Endevor mappings and build your local Git changes before synchronizing them to Endevor.

Usage

zowe endevor-bridge-for-git <group>

Where <group> is one of the following:

Groups

  • build | b - Build changes from git local repository in Endevor
  • endevor-credentials | ec - Manage the stored Endevor credentials.
  • mapping | m - Manage the Git-Endevor mappings.
  • work-area | wa - Use an Endevor work area from the list defined by your administrator.
