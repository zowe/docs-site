Create data sets, data set members or uss files and directories

Usage

zowe rse-api-for-zowe-cli create <command>

Where <command> is one of the following:

Commands

  • data-set | ds - Create a data set based on the properties of an existing data set
  • data-set-member | member - Create a member for a Partitioned data set.
  • data-set-partitioned | pds - Create partitioned data sets (PDS)
  • data-set-sequential | ps - Create physical sequential data sets (PS)
  • uss-directory | dir - Create a uss directory.
  • uss-file | file - Create a uss file.
