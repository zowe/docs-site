Perform basic NetView operations.

Usage

zowe znetview-base <group>

Where <group> is one of the following:

Groups

  • auth - Connect to the NetView REST Server authentication service and obtain a token, or disconnect from the NetView REST Server and revoke the token
  • issue - Issue NetView commands
  • list | ls - List Canzlog messages, task utilization information, and active NetView domains
  • record | rc - Record a message from Canzlog with provided information
