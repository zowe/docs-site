Download content from z/OS data sets and USS files to your PC.

Usage

zowe zos-files download <command>

Where <command> is one of the following:

Commands

  • all-members | am - Download all members from a pds
  • data-set | ds - Download content from a z/OS data set
  • data-sets-matching | dsm | Download - multiple data sets at once data-set-matching
  • uss-directory | uss-dir - Download content from a USS directory
  • uss-file | uf | uss - Download content from a USS file
