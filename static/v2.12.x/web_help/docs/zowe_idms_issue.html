Issues IDMS DCMT and DCUF commands

Usage

zowe idms issue <command>

Where <command> is one of the following:

Commands

  • dcmt-display | dd - DCMT DISPLAY commands
  • dcmt-help | dh - DCMT HELP command
  • dcmt-quiesce | dq - DCMT QUIESCE command
  • dcmt-shutdown | shutdown - DCMT SHUTDOWN commands
  • dcmt-statistics | ds - DCMT STATISTICS commands
  • dcmt-test | dt - DCMT TEST commands
  • dcmt-vary | dv - DCMT VARY commands
  • dcuf-help | dfh - DCUF HELP command
  • dcuf-show | dfs - DCUF SHOW commands
