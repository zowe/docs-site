zowe → profiles → delete

Delete existing profiles.

Usage

zowe profiles delete <command>

Where <command> is one of the following:

Commands

  • cics-profile | cics - Delete a cics profile.
  • db2-profile | db2 - Delete a db2 profile.
  • ssh-profile | ssh - Delete a ssh profile.
  • tso-profile | tso - Delete a tso profile.
  • zosmf-profile | zosmf - Delete a zosmf profile.
