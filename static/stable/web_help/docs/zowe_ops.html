Usage

zowe ops <group>

Where <group> is one of the following:

Groups

  • create - Create a generic AOF event for API rules processing.
  • disable - Disable an AOF rule.
  • enable - Enable an AOF rule.
  • execute - Execute an OPS/MVS REXX program or an operator command.
  • show - Display data associated with OPS/MVS automation elements, such as AOF rules and rulesets, and SSM resources, OPS log names, subsystems and RDF tables, or REST API server status.
  • start - Start an SSM resource.
  • stop - Stop an SSM resource.
