List instances, elements, types, packages and inventory locations in Endevor.

Usage

zowe endevor list <command>

Where <command> is one of the following:

Commands

  • code-page-config | codepages | cdpg - The list code-page-config command lists the EBCDIC code page mapping rules used by Endevor Web Services
  • defaults - List the contents of the main Endevor configuration table (C1DEFLTS)
  • dialog - List the contents of the Endevor ISPF dialog defaults table (ENDIECFG)
  • elements | elem | ele - The list elements command lists element information in Endevor
  • environments | environment | env - The list environments command lists environments in Endevor
  • features - List the contents of the Endevor optional features table (ENCOPTBL)
  • instances | instance | inst | i - The list instances command lists instances used by Endevor Web Services
  • packages | package | pkg | p - The list packages command lists package information in Endevor
  • processor-groups | processor-group - | The list processor-groups command lists
  • pgrp - processor group information in Endevor
  • processor-symbols | processor-symbol | The - list processor-symbols command lists
  • psym - processor group symbol overrides information in Endevor
  • stages | stage | stg - The list stages command lists stages in Endevor
  • subsystems | subsystem | subsys | sbs - The list subsystems command lists subsystem information in Endevor
  • symbols - List the Endevor site symbols
  • systems | system | sys | s - The list systems command lists system information in Endevor
  • tasks - List asynchronous tasks submitted by this user
  • type-sequence | typeseq - List the contents of the Endevor type sequence
  • types | type | t - The list types command lists type information in Endevor
