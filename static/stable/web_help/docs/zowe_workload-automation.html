Interact with IBM Z Workload Scheduler

Usage

zowe workload-automation <group>

Where <group> is one of the following:

Groups

  • execute | exec - Execute WAPL commands
  • get - Get resources or jobs in plan.
  • list | ls - List jobs, jobstreams, and resources in plan, list jobstreams in model(database)
  • set-status | sets - Set the status of a job in plan
  • submit | sub - Submit a jobstream into the plan
  • update | upd - Update resources or jobs in plan.
