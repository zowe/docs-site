Use the Endevor Bridge for Git plug-in to manage your Git-Endevor mappings and build your local Git changes before synchronizing them to Endevor.

Usage

zowe endevor-bridge-for-git <group|command>

Where <group|command> is one of the following:

Groups

  • add - Add
  • administrate | admin - Perform advanced actions (for administrators)
  • create | cr | c - Create mappings, branches, endevor connections, etc
  • delete | del | d - Delete mappings, branches, endevor connections, etc
  • list | ls - List mappings, branches, connections, users, processor-groups, etc
  • rename | r - Rename connection
  • transfer - Transfer mapping, owner, connection
  • update | u - Update your endevor-credentials,endevor connections, processor groups, etc

Commands

  • sync-back - Schedule sync-back
