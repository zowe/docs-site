Use the Endevor Bridge for Git plug-in to manage your Git-Endevor mappings and build your local Git changes before synchronizing them to Endevor.

Usage

zowe endevor-bridge-for-git <group>

Where <group> is one of the following:

Groups

  • branch | br - Manage the branches of your Git-Endevor mapping.
  • build | b - Build changes from git local repository in Endevor
  • endevor-connection | co - Manage the stored Endevor connections.
  • local-hook-token | token - Manage your stored local hook token
  • mapping | m - Manage the Git-Endevor mappings.
  • processor-group | proc-group | pg - Manage the processor groups of your mapping
  • work-area | wa - Use an Endevor work area from the list defined by your administrator.
