Manage JSON project and global configuration

Usage

zowe config <command>

Where <command> is one of the following:

Commands

  • auto-init - Automatically generate a config from APIML
  • convert-profiles | convert - Convert profiles to team config
  • edit - edit config files
  • import - import config files
  • init - init config files
  • list | ls - list config properties
  • profiles - displays profile paths
  • schema - dumps the JSON schema for the config
  • secure - secure configuration properties
  • set - set configuration property
  • update-schemas | us - update schema files
