Create a z/OSMF workflow on a z/OS system.

Usage

zowe zos-workflows create <command>

Where <command> is one of the following:

Commands

  • workflow-from-data-set | wfds - Create a z/OSMF workflow on a z/OS system using a Data set
  • workflow-from-local-file | wflf - Create a z/OSMF workflow on a z/OS system using a Local file
  • workflow-from-uss-file | wfuf - Create a workflow instance in z/OSMF using a USS file
