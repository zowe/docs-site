A profile to issue commands to a z/OS system with a working Zowe REST server,Mediation Layer, or IBM RSE (Remote System Explorer) API server installation.

Usage

zowe profiles list rse-profiles [options]

Options

  • --show-contents | --sc (boolean)

    • List rse profiles and their contents. All profile details will be printed as part of command output.

Examples

  • List profiles of type rse:

    • zowe profiles list rse-profiles

  • List profiles of type rse and display their contents:

    • zowe profiles list rse-profiles --sc
