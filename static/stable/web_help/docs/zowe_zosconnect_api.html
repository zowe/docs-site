Manage z/OS Connect EE APIs

Usage

zowe zosconnect api <command>

Where <command> is one of the following:

Commands

  • delete - Delete an API
  • info - Get detailed information about an API
  • install - Install a new API
  • list - List the APIs installed in the server
  • start - Set an APIs status to started
  • stop - Set an APIs status to stopped
  • update - Update an existing API
