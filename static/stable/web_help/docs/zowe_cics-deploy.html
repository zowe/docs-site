CICS bundle deployment plugin.

Usage

zowe cics-deploy <group>

Where <group> is one of the following:

Groups

  • deploy | d | dep - Deploy a CICS bundle to one or more CICS regions within a CICSplex
  • generate | g | gen - Transform the working directory into a CICS bundle
  • push | p - Push a CICS bundle from the working directory to one or more CICS regions within a CICSplex
  • undeploy | u | udep - Undeploy a CICS bundle from one or more CICS regions within a CICSplex
