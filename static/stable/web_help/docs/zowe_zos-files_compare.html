zowe ‚Ä∫ zos-files ‚Ä∫ compare

Compare the contents of z/OS data sets or USS files in your terminal (stdout).

Usage

zowe zos-files compare <command>

Where <command> is one of the following:

Commands

  • data-set | ds - Compare the contents of two z/OS data sets
  • local-file-data-set | lf-ds - Compare the contents of a local file and a z/OS data set
  • local-file-spool-dd | lf-sdd - Compare the contents of a local file and a spool dd
  • local-file-uss-file | lf-uss - Compare the contents of a local file and a z/OS USS file
  • spool-dd | sdd - Compare the contents of two spool dds
  • uss-files | uss | uf - Compare the contents of two z/OS USS files
