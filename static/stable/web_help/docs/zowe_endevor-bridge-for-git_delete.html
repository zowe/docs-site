Usage

Where <command> is one of the following:

Commands

  • branch | br - Remove a branch from an existing mapping
  • endevor-connection | connection | con - Delete existing Endevor connection
  • endevor-connection-certificate - | Delete existing certificate
  • connection-certificate | certificate - | cer
  • local-hook-token | token - Revoke your local hook token.
  • mapping | m - Delete a Git-Endevor mapping
  • mf-webhook-settings | mf-webhook - Delete MF Webhook Server.
  • user | u - Delete existing user
  • webhook-settings | webhook - Delete Webhook.
