Create a data set.

Usage

zowe file-master-plus create <command>

Where <command> is one of the following:

Commands

  • large-format-sequential | large - Create a large format sequential data set
  • like-model | model | lm - Create a data set based on the parameters of an existing data set
  • partitioned-data-set | pds - Create a partitioned data set
  • partitioned-data-set-extended | pdse - Create an extended partitioned data set
  • physical-sequential | ps - Create a physical sequential data set
  • vsam-esds | esds - Create an ESDS VSAM data set
  • vsam-ksds | ksds - Create a KSDS VSAM data set
  • vsam-lds | lds - Create a LDS VSAM data set
  • vsam-rrds | rrds - Create a RRDS VSAM data set
  • vsam-vrrds | vrrds - Create a VRRDS VSAM data set
