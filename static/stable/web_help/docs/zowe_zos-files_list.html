List data sets and data set members. Optionally, you can list their details and attributes.

Usage

zowe zos-files list <command>

Where <command> is one of the following:

Commands

  • all-members | am - List all members of a PDS
  • data-set | ds - List data sets
  • file-system | fs - Listing mounted z/OS filesystems
  • uss-files | uf | uss - List USS files
