Lists z/OSMF provisioning information such as the provisioned instances from the registry, the provisioned instance details, the available provisioning templates and provisioning template details.

Usage

zowe provisioning list <command>

Where <command> is one of the following:

Commands

  • catalog-templates | ct - List z/OSMF published catalog templates
  • instance-info | ii - List Provisioned Instance Details
  • instance-variables | iv - List Instance Variables and Values
  • registry-instances | ri - List provisioned instances
  • template-info | ti - List Published Template Details
