Welcome to the IBM RSE API Plug-in for Zowe CLI!

Usage

zowe rse-api-for-zowe-cli <group>

Where <group> is one of the following:

Groups

  • auth - Connect to RSE API Server Authentication Service
  • cancel | can - Cancel a job
  • change - Change User Data on the Host
  • check - Confirm that RSE API is running on a specified system and gather information about the RSE API server and Address Space for diagnostic purposes. Obtain and update MVS mappings file
  • copy - Copy a data set
  • create | cre - Create data sets
  • delete | del - Delete a data set or Unix System Services file
  • download | dl - Download content from z/OS data sets, USS files, and JOB Output to your PC.
  • issue - Issue TSO and UNIX commands
  • list | ls - List the details for data sets and the members in the data sets, uss files and directories, and jobs and spool files.
  • migrate | hmigr | hMigrate - Migrate data sets.
  • recall | hrec | hRecall - Recall migrated data sets.
  • rename | rn - Rename a Data set, Data set member, File or Directory
  • submit | sub - Submit z/OS jobs
  • upload | ul - Upload the contents of a file to z/OS data sets or to uss
  • view | vw - View details of a z/OS job
