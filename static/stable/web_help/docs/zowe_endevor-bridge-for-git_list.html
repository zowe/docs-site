List mappings, branches, connections, users, processor-groups, etc

Usage

zowe endevor-bridge-for-git list <command>

Where <command> is one of the following:

Commands

  • actions | ac - List actions
  • app-logs | logs - List logs
  • branch | br - List the branches of an existing mapping
  • endevor-change-metadata | endevor-change - List CCID, Comment, and From-location values
  • endevor-connection | connection | con - List Endevor connections
  • endevor-connection-certificate - | List Endevor connection certificate
  • connection-certificate | certificate - | cer
  • generate-settings | generate | gen - List generate settings
  • git-change-metadata | git-change - List the commit messages that your mapping uses for Bridge sync-back commits
  • mail-server-settings | mail-server - List Mail Server settings
  • mapping | m - List Git-Endevor mappings
  • mf-webhook-settings | mf-webhook - List MF webhooks
  • operations | o - List operations
  • processor-group | pg - List Processor Groups
  • sync-back-settings | sync-back - List sync-back (refresh)
  • user | u | users - List users
  • webhook-settings | webhook - List webhooks
