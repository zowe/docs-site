Welcome to the CA 7 Zowe CLI!

Usage

zowe ca7 <group>

Where <group> is one of the following:

Groups

  • agent | agt - Agent management
  • agent-cred | agtcred - Agent credential management
  • batch | batch - Batch API management
  • CA7-instances | inst - REST API CA 7 Instances
  • commands | cmds - All the CA 7 commands
  • dataset | ds - Dataset management
  • job-definition | jd - CA 7 job definitions
  • job-history | jh - Job History
  • job-instance | ji - All methods of a Job Instances
  • resources | res - REST API CA 7 Instances
  • transaction | trans - REST API Transaction
  • version | version - REST API Version
