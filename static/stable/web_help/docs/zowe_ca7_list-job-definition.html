All the available commands are listed in the COMMANDS section.

Usage

zowe ca7 list-job-definition <command>

Where <command> is one of the following:

Commands

  • job | j - Get basic definition for one or more jobs
  • prose | p - Get prose information for a job
  • requirement-predecessor | rp - Get predecessors for one or more jobs
  • requirement-successor | rs - Get successor jobs for one or more jobs
  • resource | r - Get job resources for one or more jobs
  • schedule | s - Get job schedules for one or more jobs
  • step-dd | sd - Get job STEP and DD information for one or more jobs
  • trigger-predecessor | tp - Get trigger objects for one or more jobs
  • trigger-successor | ts - Get triggered jobs for one or more jobs
