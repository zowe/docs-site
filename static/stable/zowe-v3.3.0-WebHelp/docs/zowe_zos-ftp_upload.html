Upload data set and USS content

Usage

zowe zos-ftp upload <command>

Where <command> is one of the following:

Commands

  • file-to-data-set | ftds - Upload from local file to data set
  • file-to-uss-file | ftuf | ftu - Upload a local file to a USS File
  • stdin-to-data-set | stds - Upload from stdin to a data set
  • stdin-to-uss-file | stuf | stu - Upload stdin to a USS File
