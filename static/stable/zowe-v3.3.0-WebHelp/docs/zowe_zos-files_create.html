Create data sets.

Usage

zowe zos-files create <command>

Where <command> is one of the following:

Commands

  • data-set | ds - Create data sets based on the properties of an existing data set
  • data-set-binary | bin - Create executable data sets
  • data-set-c | dsc - Create data sets for C code programming
  • data-set-classic | classic - Create classic data sets (JCL, HLASM, CBL, etc...)
  • data-set-partitioned | pds - Create partitioned data sets (PDS)
  • data-set-sequential | ps - Create physical sequential data sets (PS)
  • data-set-vsam | vsam - Create a VSAM cluster
  • uss-directory | dir - Create a UNIX directory
  • uss-file | file - Create a UNIX file
  • zos-file-system | zfs - Create a z/OS file system
