# @botname ims list region

List ims region.

## Usage

`@botname ims list region \*imsregionname options`

## Positional arguments

-   **\*imsregionname**

    Positional arguments are used to specify the name of resource object.


## Options

-   **--ims-id \(ii\) ims id**

    Options are used to specify the location of resource object. Options are required for this command.


## Examples

-   **`@bnz ims list region --ims-id IMSA`**

    List all region of ims IMSA.

-   **`@bnz ims list region IMSADLI --ii IMSA`**

    List the region IMSADLI of ims IMSA.

-   **`@bnz ims list region IMS* --ii IMSA`**

    List the region of ims IMSA whose name start with IMS.
