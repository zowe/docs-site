# Version 2.18.5 (January 2026)

Welcome to the Zowe Version 2.18.5 release!

See [Bug fixes](#bug-fixes) for a list of issues addressed in this release.

**Download v2.18.5 build**: Want to try new features as soon as possible? You can download the v2.18.5 build from [Zowe.org](https://www.zowe.org/download.html).

## Bug fixes

Zowe Version 2.18.5 contains the bug fixes that are described in the following topics.


### Zowe CLI

#### Zowe CLI (Core)

- Updated `tar` dependency to resolve technical currency. [#2668](https://github.com/zowe/zowe-cli/pull/2668)

#### Zowe CLI Imperative Framework

- Removed large dependency `pacote` for fetching package metadata when installing plug-ins. ([#2668](https://github.com/zowe/zowe-cli/pull/2668))
- Fixed truncation that could occur when the `AbstractRestClient` request method downloads gzip-encoded data. ([#2661](https://github.com/zowe/zowe-cli/pull/2661))
